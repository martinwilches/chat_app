# Chat App Frontend

## Packages

- __tailwindcss/vite__ :: Integrar Tailwind directamente con Vite
- __zustand__ :: Gestor de estados global, usado para compartir estado entre componentes
- __zod__ :: Validar y tipar datos, por ejemplo formularios o respuestas de API's
- __sonner__ :: Notificaciones tipo toast
- __socket.io-client__ :: Cliente para comunicacion en tiempo real con un servidor Socket.IO
- __react-router__ :: Manejo de rutas y navegacion en la aplicacion
- __react-hook-form__ :: Manejo eficiente de formularios en React. Reduce renders innecesarios, facilita validaciones.
- __@hookform/resolvers__ :: Puente entre react-hook-form y zod
- __axios__ :: cliente http para consumir API's
- __lucide-react__ :: Conjunto de iconos SVG para React
- __@tanstack/react-query__ :: Manejo avanzado de estados del servidor
