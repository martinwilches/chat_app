# Chat App Frontend

## Packages

- __tailwindcss/vite__
  - Integrar Tailwind directamente con Vite.
- __zustand__
  - Se utiliza para compartir estado entre componentes.
- __zod__
  - Define esquemas que validan datos en tiempo de ejecucion, se integra perfectamente con TypeScript.
- __sonner__
  - Notificaciones (toasts).
- __socket.io-client__
  - Comunicacion en tiempo real con un servidor Socker.IO.
- __react-router__
  - Enrutamiento en aplicaciones React.
- __react-hook-form__
  - Reduce renders innecesarios, simplifica validaciones y manejo de errores.
- __@hookform/resolvers__
  - Puentre entre react-hook-form y librerias de validacion. Basicamente conecta el formulario con el esquema de validacion.
- __axios__
  - Cliente HTTP para consumir API's.
- __lucide-react__
  - Conjunto de iconos SVG para React
- __@tanstack/react-query__
  - Gestion avanzada de estados remotos (datos que vienen del servidor). Se encarga de cachear, sincronizar, reintentar y actualizar datos de API's.
